# TL Fire Watch

SF Tenderloin fire damage finder with viability scoring for restoration services. Pulls live data from SF OpenData (DataSF).

## Setup

```bash
pnpm install
pnpm dev
```

## Data Sources

- [Fire Incidents](https://data.sfgov.org/Public-Safety/Fire-Incidents/wr8u-xric) — SF Fire Department incident records
- [Building Permits](https://data.sfgov.org/Housing-and-Buildings/Building-Permits/i98e-djp9) — DBI permit records for cross-referencing repairs
